## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/AI-NLP-Daily-Practice.git
```

2. Move into the project directory

```bash
cd AI-NLP-Daily-Practice
```

3. Install the required libraries

```bash
pip install nltk spacy transformers scikit-learn pandas numpy
```

4. Run any Python file

```bash
python filename.py
```

or open the Jupyter notebooks using:

```bash
jupyter notebook
```
